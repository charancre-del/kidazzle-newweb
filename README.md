# KIDazzle Child Care Theme

Premium WordPress theme for KIDazzle Child Care centers.

## Overview

A fully-featured WordPress theme built for child care centers, featuring:

- **Modern Design**: TailwindCSS-powered responsive design
- **Custom Post Types**: Locations, Programs, Cities, Team Members
- **SEO Engine**: Built-in SEO optimization with schema markup
- **LeadConnector Integration**: Embedded contact forms
- **ACF Integration**: Optional Advanced Custom Fields support

## Requirements

- WordPress 6.0+
- PHP 8.0+
- Advanced Custom Fields Pro (optional but recommended)

## Installation

1. Upload `kidazzle` folder to `/wp-content/themes/`
2. Activate theme in WordPress admin
3. Configure customizer settings
4. Set up navigation menus

## Menu Locations

- **Primary**: Main navigation
- **Footer Quick Links**: Footer column 2
- **Footer Resources**: Footer column 3

## Customizer Options

- Hero section content and image
- Contact information (phone, address)
- Footer tagline
- Form embed URL

## Custom Post Types

| Post Type | Description |
|-----------|-------------|
| `location` | Child care center locations |
| `program` | Age-based programs |
| `city` | City/area pages for local SEO |
| `team_member` | Staff profiles |

## Development

```bash
# Install dependencies
npm install

# Build CSS
npm run build:css

# Watch for changes
npm run dev
```
